# Heart-Attack-Prediction-Hybrid
Hybrid heart attack prediction model combining XGBoost and Artificial Neural Networks to capture both structured and nonlinear patterns in medical data. Achieves ~88.5% accuracy with strong ROC-AUC performance on UCI dataset.

This project implements a hybrid machine learning and deep learning model to predict heart attack risk using medical data.

## 📌 Overview

The model combines XGBoost and Artificial Neural Networks (ANN) to capture both structured patterns and complex nonlinear relationships in patient data.

## 🎯 Objective

* Predict likelihood of heart attack using medical features
* Improve performance using hybrid modeling
* Combine strengths of ML and deep learning

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* TensorFlow / Keras

## 🔍 Methodology

* Data preprocessing and feature engineering
* Training XGBoost model
* Building ANN model
* Combining predictions for improved accuracy

## 📈 Results

* Accuracy: ~88.5%
* ROC-AUC: ~0.93

## 📊 Dataset

UCI Heart Disease Dataset (Kaggle)
Dataset not included due to size limitations.

## 💡 Key Insight

Hybrid models combining ML and DL can outperform standalone models by capturing both structured and nonlinear relationships.
